# Flask To-Do List App

A simple To-Do List web application built using Python and Flask.  
This project is designed for beginners to understand how backend and frontend work together.

---

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Tasks are saved using a JSON file (data persists after refresh)
- Clean and user-friendly interface

---

## 🛠 Tech Stack
- Python
- Flask
- HTML
- CSS
- JSON (for data storage)

---

## 📂 Project Structure
ToDo-List/

├── app.py

---


## ▶️ How to Run the Project

1. Install Flask
   pip install flask
2. Run the application
   python app.py
3. Open your browser and visit
   http://127.0.0.1:5000
